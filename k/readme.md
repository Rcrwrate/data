# 崩坏学园里的kiana---可爱的页面精灵插件
# [源地址](https://github.com/51itclub/kiana)
使用方式:
```javascript
<div id="div1"></div>
   
<script src="https://d.sirin.xyz/k/src/js/jquery.min.js"></script>
<script src="https://d.sirin.xyz/k/src/js/kiana.js"></script>
<script>
   $("#div1").KianaInit({
         randomPosition:true
    });
</script>
```
